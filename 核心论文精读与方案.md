## 第二部分：4 篇精读要点卡

### #1 Reject or Not（arxiv 2512.10257，美的，拒识）
- **核心**：首个中文智能家居拒识 benchmark（11,913 对，13 类）+ 三层 LLM 拒识（Qwen-2.5-3B adapter + 家庭历史对话 + RAG bad case），加权准确率 **96.75%**。
- **关键结论**：拒识是语义判断；小模型 + prompt + SFT（96.44%）> GPT-5 零样本（~90%），且对效率友好。
- **13 类拒识场景**：非人声误触发、歧义指代、设备无关请求、闲聊、ASR 错乱短句…（极可能是测试集拒识样本类型）。
- **局限**：语音子集是 TTS 合成、依赖 ASR 前端、重叠覆盖薄、指标只有 accuracy（需自算拒识率）。
- **复用**：拒识基座 Qwen-2.5-3B SFT + RAG bad case 闭环。

### #2 ASE-PVAD（arxiv 2601.12769，东华×美的，目标检测）
- **核心**：短 enrollment（唤醒词级 0.5s）Personal VAD。**洞察：目标声纹藏在混合音频里**——从混合音频按相似度选目标关键帧声纹，加性融合反哺 enrollment，长时迭代更新（λ=0.1 残差锚定），**5 次迭代追平 7.4s 全注册，不增参数**。
- **关键数字**：0.5s 短注册 REC 58.94→68.12，TSS AP 75.43→83.38。
- **结构**：CAM++（192d）声纹 + 4 层 Conformer + FiLM 调制 → 帧级三分类（NS/NTSS/TSS）。
- **对应本题**：TSS 帧送 ASR（只转写目标）、NTSS 帧拒识——天然覆盖 CER+拒识。
- **风险**：论文 SNR 0–20dB（本题 −5dB 需下扩）；高重叠单帧选择需升级为 Top-K 加权。

### #3 美的 KWS 空间先验（arxiv 2603.09505，美的）
- **核心**：端到端多通道 KWS。spatial encoder 学通道间 IPD/ILD + spatial embedding 注入 DOA 离散先验 + MDTC 流式 backbone（279k 参数）。
- **关键结论**：端到端 > cascaded（2ch E2E 76.92%@0dB > GSC 级联 72.19%）。
- **致命前提**：**完全依赖多通道**，单通道核心创新失效；未开源；用 oracle DOA；SNR 只测到 0dB。
- **价值**：坐实"端到端反 cascaded"立场 + MDTC 流式 backbone 可借。

### #4 DSENet（arxiv 2507.20926，美的×东南大学，DOA 提取）
- **核心**：端到端定向语音提取。DOA + beamwidth 双软约束 + cyc-pos 方向编码（40d 替代 360 one-hot，1.40M 参数）。
- **关键数字**：**6 人重叠**，3 通道圆阵 r=30mm；2spk WER 10.52%、3spk 17.31%（vs Noisy 82%/96%）；MVDR 完全失效。
- **拒识巧思**：Stage2 训练 10% 空扇区样本→无目标输出近静音→ASR 置空即拒识。
- **单通道退化**：Crossband/Narrowband 主干 + 把 DOA embedding 换成声纹 embedding（Clue Encoder element-wise 乘）= 声纹引导 TSE。
- **开源**：github.com/jingkangqi/DSENet。

## 第三部分：综合技术方案骨架（基于 4 篇）

```
唤醒音频(enrollment)
   │ CAM++ 抽 192d 声纹 Eenroll  [+ ASE 自增强：从识别音频选目标关键帧反哺]
   ▼
识别音频 ──► [多通道时: DSENet DOA锁方向提取] ──► [单/多通道: Personal VAD 帧级三分类]
                                                        │
                              ┌─────────────────────────┼─────────────────────────┐
                              ▼ TSS(目标帧)              ▼ NTSS/NS(非目标)        ▼ 空扇区/低置信
                        送 ASR 转写                 拒识(声纹级)             拒识(空间/置信级)
                              │                                                  
                              ▼ FireRedASR-AED + 家居指令 LM 约束解码
                              │
                              ▼ Qwen-2.5-3B LLM 语义拒识(判 query 是否有效)  ← RAG bad case
                              ▼
                    输出文本 / 空字符串(拒识)
```

**三大考点对应**：
- CER 40%：PVAD 只对 TSS 转写 + FireRedASR-AED + 家居 LM 约束
- 拒识 40%：三层拒识（PVAD NTSS + 空扇区静音 + Qwen LLM 语义）
- 效率 20%：全轻量组件（KWS 279k / DSENet 1.4M / PVAD 4层Conformer / Qwen-2.5-3B）
